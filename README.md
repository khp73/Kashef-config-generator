# کاشف ۱۱ (Kashef 11) - تولید و اهدای کانفیگ V2Ray به MahsaNG

## 🌟 معرفی
کاشف ۱۱ یک ابزار قدرتمند مبتنی بر وب است که به صورت خودکار از **۹ منبع معتبر** گیت‌هاب، کانفیگ‌های V2Ray را دریافت، تست و دسته‌بندی می‌کند. با استفاده از آن، کاربران MahsaNG می‌توانند لینک سابسکریپشنی دریافت کنند که همیشه به‌روز است و مشکل کمبود کانفیگ را برای همیشه حل می‌کند.

## ✨ ویژگی‌های کلیدی
*   **دریافت خودکار از ۹ منبع معروف** مانند Nexus-Nodes و Freedom-V2Ray.
*   **فیلتر جغرافیایی هوشمند** (آسیا، اروپا، آمریکا و...).
*   **تست پینگ واقعی** و نمایش کانفیگ‌های VIP بر اساس کمترین زمان پاسخ.
*   **خروجی سابسکریپشن استاندارد** (Base64) برای استفاده آسان در MahsaNG.
*   **امکان دانلود کانفیگ‌ها** در قالب فایل‌های TXT، CSV و JSON.

## 🚀 راهنمای استفاده در MahsaNG
1.  اپلیکیشن MahsaNG را باز کنید.
2.  از منوی سمت چپ (☰)، گزینه **«Get EMS Config»** را انتخاب کنید.
3.  لینک زیر را در کادر مربوطه کپی کرده و روی **«Add»** کلیک کنید:
    `https://[Your-Username].github.io/[Your-Repo-Name]`
4.  حالا می‌توانید سابسکریپشن اضافه شده را انتخاب کرده و به اینترنت آزاد متصل شوید.
